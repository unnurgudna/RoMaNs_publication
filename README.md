# RoMaNs_publication

### Code used for RoMaNs (restoration of microbiota in infants) strain tracking publication

📊 Analysis performed by Unnur Guðnadóttir, Stefanie Prast-Nielssen and Elin Rohdin during 2025-2026

🧪 Fecal sample analysis published, with the same approach used for other sample types.

#### Study design:

📝 Intervention study, comparing microbiome samples from women and infants delivered:
1) Vaginally
2) Elective C-section with microbiome restoration intervention (CS-I)
3) Elective C-section with placebo (CS-P)

🖥️ More information: https://tinyurl.com/37etkc7j

#### 💡 Workflow order:
- Descriptive table
- Decontamination
- RA filtering
- CLR transformation
- Beta diversity
- Alpha diversity
- Maternal comparison
- Temporal analysis (TEMPTED)
- Strain tracking (samestr)
